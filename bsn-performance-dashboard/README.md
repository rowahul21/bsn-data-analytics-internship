# 📈 BSN 2025 Realization Performance Dashboard

## 📊 Overview
This dashboard provides a comprehensive evaluation of financing performance in 2025, enabling stakeholders to monitor achievement and identify performance gaps across branches.

## 🎯 Objective
To simplify annual performance analysis and support strategic evaluation through centralized and interactive reporting.

## 👥 Users
- Bank Management
- Branch Offices

## 📂 Data Source
- Internal Bank Data
- Data types:
  - Time series
  - Transaction data
  - Realization metrics

## 🛠️ Tools & Skills
- Google Looker Studio
- SQL
- Python
- Excel / Spreadsheet
- Data Cleaning & Transformation

## 🔗 Dashboard Link
https://datastudio.google.com/reporting/6a87fb01-8f43-4696-848b-47575d0642a7

## 📈 Key Insights
- Financing realization increased by **45% YoY**, from IDR 9.67B (2024) to IDR 14.07B (2025)
- Performance breakdown highlights variation across branches in both realization and outstanding position
- Ticket size analysis reveals efficiency differences:
  - Some branches achieve high realization with fewer units (efficient strategy)
  - Others rely on higher volume (high effort approach)

## 💡 Business Value
This dashboard enables management to evaluate branch performance more effectively, identify high-performing units, and optimize strategic resource allocation.
